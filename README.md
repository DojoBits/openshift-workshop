# 👋 Welcome

An enterprise hands-on workshop focused on operating applications on **Red Hat
OpenShift** using the **GitOps operational model**.

Participants deploy and manage applications on OpenShift using **OpenShift
GitOps (Argo CD)** while learning how platform teams maintain consistent and
automated environments where **where Git serves as the single source of
truth**.

The workshop demonstrates real operational workflows used in modern
**OpenShift-based Kubernetes platforms**.

## 🚀 Workshop Highlights

- Deploy applications on **Red Hat OpenShift**
- Implement **GitOps workflows with OpenShift GitOps (Argo CD)**
- Manage application lifecycle using **Git as the source of truth**
- Observe **automated reconciliation and drift detection**
- Understand operational practices used in **enterprise OpenShift platforms**

# 🛠 Workshop Environment

The workshop environment is based on:

- **Red Hat OpenShift**
- **OpenShift Local (CRC)**
- **OpenShift GitOps Operator**
- **Argo CD**

# 📋 Prerequisites

To successfully complete the workshop you will need:

- Web browser
- SSH client
  Examples:
  - OpenSSH
  - MobaXterm
  - PuTTY
- Internet connectivity to the lab environment
- Local administrator privileges on your machine
